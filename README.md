# 🧊 灵犀魔方 (Sentient Cube)

> **懂记忆、更有“生命”的具身智能桌面伴侣**

## 📖 项目简介 (Introduction)

**灵犀魔方** 是一款具身智能（Embodied AI）家居终端。它摒弃了传统屏幕的冷漠感，采用“隐形科技 (Shy Tech)”设计理念，将 AI 的智慧封装在一个会呼吸、会变形、能看见物理世界的织物立方体中。

它既是高效的“第二大脑”，也是温暖的“情绪光雕塑”，更是拥有空间视觉的“寻物猎犬”。

---

## 🛠 核心硬件设计 (Hardware & Design)

我们采用了“极简外表 + 机械仿生内核”的设计方案，确保量产性与独特的交互美学。

### 1. 外观材质
* **隐形设计**：采用高透声、透光的高级灰/燕麦色声学织物包裹。
* **触感**：外观如同一块温润的鹅卵石或木块，无电子产品侵入感。

### 2. 交互界面
* **幻影点阵屏 (Phantom LED Matrix)**：隐藏在织物下方的 LED 矩阵。平时不可见，唤醒时透过织物显示像素化表情、时间或状态，呈现朦胧的科技美学。
* **仿生机械云台 (Bionic Gimbal)**：顶部的隐藏式升降结构。内置“潜望镜”式镜头，模拟生物“转头看”的动作，提供极强的生命感 (Animism)。

### 3. 动力骨架
* **呼吸律动**：内部微型舵机组驱动外壳板块进行微小的开合运动，模拟生物的呼吸。

## 🧠 交互哲学：双脑协同 (Dual-Brain Synergy)

魔方模拟人类的左右脑机制，拥有两种截然不同的生存状态：

| 模式 | 🟢 右脑模式：感性伴侣 (Ambient) | 🔵 左脑模式：理性助理 (Focus) |
| :--- | :--- | :--- |
| **触发场景** | 待机、听音乐、独处、聚会 | 语音唤醒、放置物品、任务提醒 |
| **物理状态** | **“呼吸”架构**：外壳随算法节奏缓慢张合，透出呼吸灯效 | **“锁定”架构**：呼吸停止，外壳瞬间收紧闭合，云台升起 |
| **感知能力** | 识别环境噪音、光线强弱及人员动态 | 高效处理指令，点阵屏清晰显示状态 |
| **体验** | 像一只趴在桌上的猫，随音乐律动或发出温暖光晕 | 变身为干练的秘书，显示 "REC" 或 "TODO" |

## 🔦 核心功能：空间锚点 Plus (Spatial Anchor Plus)

利用顶部的 **仿生机械云台** 与 **VLM (视觉语言模型)**，灵犀魔方解决了用户“健忘”的痛点，拥有了“空间记忆”。

### 工作逻辑
1.  **视觉记忆 (Visual Mapping)**：持续扫描并理解桌面物体（如钥匙、身份证、药盒）。
2.  **智能寻物 (Smart Retrieval)**：
    * **视野内**：云台迅速旋转，发出一束**高亮光束 (Spotlight)** 精准照在物体上，并显示 "HERE ↘"。
    * **视野外**：调取语音记录进行语义回溯（例如：“你昨晚说放在了玄关抽屉”）。
3.  **纠错机制**：**视觉 > 记忆**。如果用户记错了（例如以为在抽屉里），但魔方看到物体在桌角，它会直接打光指向桌角并纠正用户。

## 🎬 用户体验剧本 (User Journey)

* **场景一：归家仪式** 🏠
    * 当你随手把车钥匙扔在桌上，魔方会通过云台微微转动确认，并在后台标记 `Key_Location = Desk_Zone_A`。
* **场景二：早晨的意外** ☀️
    * 你设定了提醒：“明早9点提醒我带身份证，在书包里”。
    * 但在8:50，你把身份证拿出来放在了桌子上。
    * **9:00 触发提醒时**：魔方检测到位置变化，云台升起并**打光照亮桌上的身份证**，大声提醒：“别忘带身份证！它在桌子上（而不是书包里）”。

## 🏁 总结 (Conclusion)

灵犀魔方不是为了取代手机，而是为了**让空间具有生命**。

它用“呼吸”安抚焦虑，用“隐形”融入家居，用“光束”指引视线。它是 AI 时代的指南针，也是桌面上最忠实的守护者。
