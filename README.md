# 🧊 灵犀魔方 (Sentient Cube)

> [cite_start]**懂记忆、更有“生命”的具身智能桌面伴侣** [cite: 1, 2]

[cite_start]灵犀魔方是一款摒弃传统屏幕冷漠感，采用“隐形科技 (Shy Tech)”设计理念的具身智能（Embodied AI）家居终端。它将 AI 的智慧封装在一个会呼吸、会变形、能看见物理世界的织物立方体中。 [cite: 4, 5]

[cite_start]它既是高效的**“第二大脑”**，也是温暖的**“情绪光雕塑”**，更是拥有空间视觉的**“寻物猎犬”**。 [cite: 6]

## 🛠️ 核心硬件设计 (Hardware & Design)

[cite_start]采用“极简外表 + 机械仿生内核”的工业设计方案，确保独特的交互美学。 [cite: 8]

* [cite_start]**🪨 表面材质**：采用高透声、透光的高级灰/燕麦色声学织物包裹。外观如温润的鹅卵石或木块，无电子产品侵入感。 [cite: 9]
* [cite_start]**✨ 交互界面 A - 幻影点阵屏 (Phantom LED Matrix)**：隐藏在织物下方。唤醒时透过织物显示像素化表情、时间或状态；关机时完全不可见。 [cite: 10, 11]
* [cite_start]**👁️ 交互界面 B - 仿生机械云台 (Bionic Gimbal)**：隐藏式升降结构。执行视觉任务时，升起“潜望镜”式镜头并旋转，模拟生物“转头”动作，赋予产品极强的**生命感 (Animism)**。 [cite: 12-14]
* [cite_start]**🫁 动力骨架**：内部微型舵机驱动外壳板块微小开合，模拟生物的**“呼吸律动”**。 [cite: 15, 16]

## 📍 核心功能：空间锚点 Plus (Spatial Anchor Plus)

[cite_start]利用 **仿生机械云台** 与 **VLM (视觉语言模型)**，解决用户“健忘”痛点，让设备获得“空间记忆”。 [cite: 31]

### 1. 视觉记忆 (Visual Mapping)
[cite_start]摄像头持续扫描并理解桌面物体，不仅识别像素，更理解语义（如“这是钥匙”、“这是药盒”）。 [cite: 33, 34]

### 2. 智能寻物逻辑 (Smart Retrieval)
[cite_start]当用户提问：“我的身份证在哪？” [cite: 36]

* **情境 A：视野内（机械指引）**
    * [cite_start]魔方云台迅速旋转，发出一束**高亮光束 (Spotlight)** 精准照在物体上，并显示 "HERE ↘"。 [cite: 37-40]
* **情境 B：视野外（语义回溯）**
    * [cite_start]调取历史对话，语音反馈：“视线内未找到。但你昨晚说放在了‘玄关抽屉’里。” [cite: 41-44]
* **情境 C：纠错机制（决策：视觉 > 记忆）**
    * [cite_start]如果用户以为在抽屉，但魔方看见你随手放在了桌角，它会直接打光指向桌角并纠正：“看，它就在这儿。” [cite: 45-47]

## 🎭 用户体验剧本 (User Journey)

### 场景一：归家仪式 🏠
* **动作**：晚上7点进屋，随手把车钥匙扔在桌上。
* [cite_start]**反应**：魔方处于呼吸模式，但云台微微转动确认，后台自动标记：`Key_Location = Desk_Zone_A`。 [cite: 49-51]

### 场景二：睡前设定 🌙
* **动作**：嘱咐魔方“明早9点提醒我带身份证，在书包里。”
* [cite_start]**反应**：屏幕显示月亮图标，确认任务。 [cite: 52-54]

### 场景三：早晨的意外 (智能判断) ☀️
* **动作**：早上8:50，你把身份证从书包拿出放在桌上，然后去洗漱。
* **触发**：9:00 提醒时间到。
* **智能行为**：系统原计划播报“在书包里”，但 VLM 检测到身份证 **此刻** 位于桌上。
* [cite_start]**最终反应**：云台升起，打光照亮桌上的身份证，大声提醒：“别忘带身份证！它在桌子上（而不是书包里）。” [cite: 55-59]
